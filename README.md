# 6-DOF Robot Arm Simulator

A browser-based robotics simulator built with React, TypeScript, and Three.js.

## Project Goals

This project explores robotic arm control and inverse kinematics through simulation.

Development roadmap:

- [x] 3-DOF Neural Network IK Prototype
- [ ] 6-DOF Three.js Simulator
- [ ] Forward Kinematics Engine
- [ ] Analytical / Numerical IK Solver
- [ ] Dataset Generation Pipeline
- [ ] Neural Network IK Approximation
- [ ] Interactive Web Deployment

---
